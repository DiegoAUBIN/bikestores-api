🔐 Authentication & Middleware

Session Authentication: Required for authenticated endpoints. Session ID stored in PHP $_SESSION after login.

API Key Authentication: Required for write operations (POST, PUT, DELETE). Header: X-API-KEY

Role-Based Access Control (RBAC):

  • employee: View own profile and showroom data only
  • chief: Manage store and staff, upload products (limited to own store)
  • it: Full administrative access to all data
